# macrosTitux

Gestionnaire de macros Bash a l'interface style MacroDroid

## Description

Application Python/Tkinter permettant de creer visuellement des declencheurs,
actions et contraintes qui seront compiles en scripts Bash executables.

## Installation

    sudo apt install python3-tk libxml2-utils
    chmod +x src/macrosTitux.py
    ./src/macrosTitux.py
